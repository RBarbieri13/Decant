## Component Specifications

### Dimensions
- **Modal Width**: 520px (max-width: 90vw for mobile)
- **Modal Max Height**: 80vh
- **Padding**: 24px
- **Border Radius**: 12px
- **Input Height**: 48px

### Colors (matching Decant theme)
| Element | Color |
|---------|-------|
| Background | `#ffffff` |
| Overlay | `rgba(0, 0, 0, 0.5)` |
| Border | `#e5e5e5` |
| Input Border | `#d4d4d4` |
| Input Border Focus | `#2d5b47` (forest green) |
| Primary Button | `#22c55e` (bright green) |
| Primary Button Hover | `#16a34a` |
| Cancel Button | `#ffffff` with `#d4d4d4` border |
| Error | `#dc2626` |
| Success | `#16a34a` |
| Muted Text | `#6b7280` |

### Typography
| Element | Size | Weight |
|---------|------|--------|
| Title | 18px | 600 |
| Label | 14px | 500 |
| Input | 14px | 400 |
| Hint Text | 12px | 400 |
| Button | 14px | 600 |

### States
1. **Empty** - Import button disabled, placeholder shown
2. **URL Entered** - Validation indicator, Import enabled
3. **Processing** - Progress steps, all inputs disabled
4. **Success** - Preview card, "Add Another" option
5. **Error** - Error message with retry option

### Keyboard Shortcuts
- `Enter` - Submit (when valid URL present)
- `Escape` - Close dialog (when not importing)
- `Cmd/Ctrl + V` - Auto-paste from clipboard

### Accessibility
- Focus trap within modal
- Escape key closes
- Auto-focus on URL input
- ARIA labels on all interactive elements
- Progress announcements for screen readers
